Introduction

chemosensR aims to analyze data from different protocols of sensory analysis -especially providing from TimeSens software ,but not only-. This vignette is dedicated to sensory profiling.

library(CSUtils)
library(chemosensR)

Reading a profile object

profileObject=profileReadData(file="../data/profile.csv",allData=NULL,outputsCsv=NULL,
                              replaceNA="crossmean",sep=";",
                              transformSessionIntoReplicate=TRUE,
                              percentageOfSubjectsByReplicate=0, 
                              percentageOfReplicatesBySubject=0,percentageOfProductBySubject=0)
